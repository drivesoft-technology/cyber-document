Getting Started

ยินดีต้อนรับเข้าสู่ Cyber Framework เวอร์ชั่น 1.0.0 เป็น CSS สำหรับตกแต่งเว็บไซต์ให้มีสีสันสดใส สวยงาม น่าใช้งาน น้ำหนักเบา และ ใช้งานง่าย เพียงแค่นำเข้าไฟล์ cyber.min.css เข้ามาในเว็บไซต์ เท่านั้น



Cyber CDN

วิธีการนำเข้าไฟล์ cyber.min.css ผ่านบริการ CDN ของ https://unpkg.com เข้ามาในเว็บไซต์ของเรา 

<link rel="stylesheet" href="https://unpkg.com/cyber-framework@1.0.0/styles/cyber.min.css" />
<link rel="stylesheet" href="https://unpkg.com/cyber-framework@1.0.0/styles/cyber-component.min.css" />


Basic template

ตัวอย่าง: การใช้งาน Cyber framework เวอร์ชั่น 1.0.0 เริ่มต้นใช้งาน Cyber framework เท่านั้น 

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8" />
        <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1" />
        <meta name="viewport" content="width=device-width, initial-scale=1, maximum-scale=1, user-scalable=no">
		<title> CYBER FRAMEWORK v1.0.0 </title>
		<!-- Cyber Framework -->
		<link rel="stylesheet" href="https://unpkg.com/cyber-framework@1.0.0/styles/cyber.min.css" />
		<link rel="stylesheet" href="https://unpkg.com/cyber-framework@1.0.0/styles/cyber-component.min.css" />
	</head>
    <body>
		<div class="container">
			<div class="row">
				<div class="col-15">
					<h1> Hello, world! </h1>
				</div> <!-- col-15 -->
			</div> <!-- row -->
		</div> <!-- container -->
	</body>
</html>
